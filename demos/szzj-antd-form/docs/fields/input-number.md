# InputNumber

数字输入框。

## features

- antd 功能
- 扩展关联校验规则 rules，支持：
  - greaterThen 填写数值大于另一个指定字段
  - greaterThenEqual 填写数值大于等于另一个指定字段
  - lessThen 填写数值小于另一个指定字段
  - lessThenEqual 填写数值小于等于另一个指定字段
