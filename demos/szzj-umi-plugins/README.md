# @szzj/umi-ones 数据部前端工具箱

## 安装依赖并启动

```bash
# 清除依赖
cnpm install lerna -g
pnpm run clean

# 安装
pnpm run install

# 移除依赖
pnpm run clean

# 打包
pnpm run build

# 调试
cd packages/template-h5
pnpm run start
```
