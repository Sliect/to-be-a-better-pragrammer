# @szzj/hooks

## ✨ Features

* useFetch 远程处理。
* useForm 处理表单。
* useList 处理列表。
* useModal 处理模态框。
* useTable 处理表格。
* useTree 处理树形数据。
* useRoutes 过滤路由数据。
* 基于 tree-shaking 按需加载。

## 📦 Install

```
cnpm i @szzj/hooks --save
```

## 🔨 Usage

```
import { useFetch } from '@szzj/hooks';
```
