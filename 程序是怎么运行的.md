# 程序是怎么运行的

## CPU

CPU 内部由寄存器、控制器、计算器、时钟四部分构成  

| 寄存器种类 | 功能 |
| ---       | ---  |
| 累加寄存器 | 存储执行运算的数据和运算后的数据 |
| 标志寄存器 | 存储运算处理后的CPU状态 |
| 程序计数器 | 存储下一条指令所在内存的地址 |
| 基址寄存器 | 存储数据内存的起始地址 |
| 变址寄存器 | 存储基址寄存器的相对地址 |
| 通用寄存器 | 存储任意数据 |
| 指令寄存器 | 存储指令 |
| 栈寄存器   | 存储栈区域的起始地址 |

程序计数器决定程序流程  

| 机器语言指令 | 功能 |
| ---         | --- |
| 数据转送指令 | 寄存器和内存、内存和内存、寄存器和外围设备之间的数据读写操作 |
| 运算指令 | 用累加寄存器执行算数运算、逻辑运算、比较运算和移位运算 |
| 跳转指令 | 实现分支、循环、强制跳转等 |
| call/return指令 | 函数的调用/返回调用前的地址 |
