// 2. npm i -S eslint

// 7. 配置 .babelrc 
// 8. npm install -D webpack webpack-cli webpack-dev-server babel-loader
// 9. 配置 webpack.config.js
// 10. 服务器上部署gitlab 和 jenkins      https://blog.csdn.net/weixin_43155926/article/details/83033371
